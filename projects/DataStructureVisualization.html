Data Structure Visualization


Codigo Fonte : Publico

Projeto de desenvolvimento proposto pelo professor Thiago Maritan para o auxiliar em sua disciplina no ensino de estrutura de dados para alunos de computação, de forma a facilitar seu entendimento dos algoritmos, mostrando de forma visual suas aplicações em um programa desktop.

Contributions

Nesse projeto eu atuei em todas as partes do projeto, desde o Design das telas a criação desses algoritmos em Java, de maneira que fosse possível relacionar, por exemplo, os nós de uma arvore criada em memória primária, com os Assets que o representava visualmente.

Métodos de estrutura de dados

Para a definição dos métodos de estrutura de dados em nossa aplicação, utilizamos das implementações já feitas na linguagem Java e as modificamos de forma que para cada posição de memória criada em uma das estruturas, houvesse um atributo gráfico contendo as propriedades RGB dos assets a serem renderizados, além de métodos para os manipular.

Visualização dos métodos

Com as estruturas de dados modificadas de forma a terem atributos gráficos e métodos para manipular essas propriedades, tivemos apenas que organizar o posicionamento dos objetos e sua renderização em uma aplicação desktop com o uso do libgdx.
