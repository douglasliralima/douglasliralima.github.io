Numerical Calculus


Codigo Fonte : Publico

Projeto de desenvolvimento proposto pelo professor Gustavo Peixoto para o auxiliar em sua disciplina no ensino de calculo númerico para alunos de computação, de forma a gerar interesse por eles, mostrando nessa aplicação o uso direto dos métodos ensinados em sua aula.

Contributions

Nesse projeto eu fiquei responsável pela criação de um servidor Django, capaz de integrar os scripts feitos em python para o calculo de equações usando os métodos númericos com paginas html desenvolvido com o auxilio do Bootstrap.

Métodos Numéricos

Implementação dos métodos do binômino de newton(...) usando as bibliotecas do numpy array e scipy, de forma a retornar um conjunto de valores com as iterações feitas pelos métodos procurando convergir no resultado correto da equações passadas, a partir de ponto iniciais.

Design de páginas

A definição estrutural das páginas foram feitas usando o Bootstrap, framework de html, css e JavaScript que facilita a criação de páginas, provendo vários elementos para serem usadas em nossa página, sendo eles customizaveis.

imagens

Paginação Dinamica com servidor Django

A partir do pressuposto do processamento pesado que alguns métodos de calculo numérico em suas iterações até sua convergência, optamos pelo método de criação dinâmica de páginas em backend a partir de um servidor Django usando o protocolo HTTP.

imagem

Com o auxilio do Jinja, deixamos pontos chaves em nossos templates, de forma a esses pontos serem modificados construidos a partir de requisições get ou post por parte do cliente para fazer certas ações no frontend. Com isso nos verificamos aquela requisição e construiamos uma página, usando o python com o Jinja provido junto do Django, para modificar a pagina, alterando o template.
